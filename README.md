# Interview-Questions-


1:- What is the difference between HTML, CSS, and JavaScript?

HTML, CSS, and JavaScript are the three fundamental technologies used to create modern websites and web applications.
HTML (HyperText Markup Language) provides the structure and content of a webpage, such as headings, paragraphs, images, links, and forms.
CSS (Cascading Style Sheets) is responsible for the visual styling and layout of the webpage, including colors, fonts, spacing, responsiveness, and animations.
JavaScript adds interactivity and dynamic functionality to the webpage, enabling features such as form validation, DOM manipulation, event handling, API integration, animations, and real-time content updates.

Simple Analogy
Think of a website as a house:
HTML = The structure and foundation of the house (walls, doors, rooms)
CSS = The design and decoration (paint, furniture, lighting)
JavaScript = The functionality and automation (switches, appliances, security systems)

Interview One-Liner
HTML creates the structure of a webpage, CSS styles its appearance, and JavaScript adds interactivity and dynamic behavior, allowing users to interact with the website effectively.
